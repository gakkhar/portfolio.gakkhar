## About

This is my personal portfolio — showcasing my work as a developer, brand communicator, and entrepreneur. Built from scratch as a single-file site with custom animations, a live GitHub repos feed, and a freelance inquiry form.

---

## Features

- Animated loading screen
- Custom cursor with lag ring effect
- Scroll-triggered reveal animations
- Live GitHub repositories (fetched via GitHub API)
- Smooth scroll navigation
- Freelance inquiry form (mailto-based)
- Fully responsive

## Deployment

Hosted via **GitHub Pages**.

To deploy your own fork:
1. Clone or download the repo
2. Rename the HTML file to `index.html`
3. Push to a repo named `<your-username>.github.io`
4. Go to **Settings → Pages → Source: main branch**
5. Live in ~2 minutes
